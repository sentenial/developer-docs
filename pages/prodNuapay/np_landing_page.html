Introduction

The Nuapay suite of RESTful APIs allow you to work with Direct Debit and Credit Transfer payments. Understand how mandates work and how to easily sign up new and existing customers. Create single direct debit or create a schedule of recurring payment or add a mandate and payment in a single API request. Various options are available.

Need to pay a supplier? See our powerful set of Credit Transfer APIs.
